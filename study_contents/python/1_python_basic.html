文字列の連結とデータ型の変換

連結は特徴があるなと思った。連結方法は2つある。
1. "+"(プラス) で連結：print('A' + 'B')
2. ","(コンマ) で連結：print('A', 'B')
違いは2点、
1つ目は","での連結では文字の間にスペースが出力されること、
2つ目は"+"の連結の場合、違う型同士の連結はerrorになるので型を統一するようキャストする必要があること。
ちなみに","での連結なら、違う型同士でも問題なく出力される。
以下コードで確認していく。

practice.py 

# 文字列の連結

print('Hello' + ' ' + 'everyone')  # 2つの文字列『Hello 』と『everyone』を連結

name = 'Tom'
print('Hello' + ' ' + name)        # 文字列『Hello 』と変数『name』を連結

print('Hello', name)         # コンマ「,」で区切って連結。自動で連結の間にスペースが入る

#------------------------------#

# データ型の変換(キャスト)
print(5 + 7)      # 数値「5 + 7」の計算
print('5' + '7')  # 文字列「5」と「7」の連結
print('5' , '7')  #コンマで連結するとスペースが入る

price = 100
print('りんごは' + str(price) + '円です')
# 文字列「りんごは」「円です」と、数値「price」の連結はできないのでpriceを文字列に変換
# print('りんごは' + x + '円です。') ←これを実行するとエラーになる

x = '10' # ''で囲んでいるので文字列
print(3 * int(x)) # x = '10'を数値(整数)に変換して計算
# print(3 * x) ←これも実行するとエラーになる

x = '0.5'
print(float(x) * 3) # 変数「x」を数値(浮動小数点数)に変換

point = 100
print('テストは', point, '点でした')
#「テストは」「点でした」は文字列。「point」は数値(整数)。
# でもコンマで区切れば、データ型を気にせず連結できる。

条件分岐

条件分岐のPointは以下のとおり。
・条件の後に”:”(コロン)をつける、elif / elseも同様に文末は":"で締める
・条件文(if文)の中の処理は、Tabでインデントをつけて表す
・否定の書き方は2種類、"x!=y" または not(x==y)
以下、条件分岐のコード例を確認する。

practice.py 

# 比較演算子

if x != 11:
  print('xは11ではない。')

if x > 5:
  print('xは5より大きい。')

if x < 11:
  print('xは5より小さい。')

if x >= 10:
  print('xは10以上。')

if x <= 10:
  print('xは10以下。')

# 条件分岐の構文

# 入力を求める。入力された値は『文字列』となるので注意！
str_score = input('点数を入力してください。')
score = int(str_score)      # 数値に変換。
str_avr = input('平均点を入力してください。')
avr = int(str_avr)

# if文の条件文とelseの末尾に”:”をつける
# 先頭に『Tab』でスペースを入れると『ifの中』、開けないと『ifの外』となる

# 変数『score』の値が80以上ならば…(80 < x)
if score > 80:
  print('よくできました。')  # 「よくできました」と表示。

# そうでないとき、変数『socre』の値が50以上または平均点以上ならば
elif (score > 50) or (avr < score):
  print('まずまずです。')    # 「まずますです。」と表示。

# そうでなければ…(x <= 50)
else:
  print('残念でした。')      # 「残念でした」と表示。
print('点数報告は以上です。') # 先頭に『Tab』でスペースを空けないと『ifの外』となるので、絶対実行される

# 反復処理(for)の構文

# 条件分岐と同様、文末に':'を忘れずに！

  print(i)          # 『i』を表示。

# 反復処理(while)の構文

# n以下の自然数の和 S = 1 + 2 + 3 + … + n において、
# S > 100 となる最小の自然数nを求めるプログラム

sum = 0
n = 1

while sum <= 100:   # sum>100になったら繰り返し終了。
  sum += n          # n = 14のとき、1+2+3+…14 = 105で100を超える
  n += 1            # 105の後でもnに1加算する処理は実行されるのでn=15になる

# 自然数が100未満となる最小の自然数は1を加算する前の14なので、nを1減らして表示。
print('S = 1 + 2 + 3 + … + n において、S > 100となる最小の自然数nは、n = ', + n - 1)  

#-----forで書くと以下のとおり-----#

sum = 0

for n in range(100):  # i = 100まで繰り返す
  sum += n            # n = 14のとき、1+2+3+…14=105になる
  if sum > 100:       # sum>100のとき、
    break             # forの繰り返しを終了

# forでは処理の文末ではなく、繰り返し処理の前に1加算しているので、そのままnを表示。
print('S = 1 + 2 + 3 + … + n において、S > 100となる最小の自然数nは、n = ', + n)

# 偶数の後ろには「〇」を表示するプログラム

# コードが長いので推奨しない処理

# 【注意】

for i in range(21):      # 0から始まるので反復回数は21にすると数字20まで反復する

配列(list)

主な特徴は以下のとおり。
・要素はコンマ「,」で区切る。
・『len(リスト名)』で、リストの長さ(要素の個数)を表示
listのコード例は以下。

practice.py 

# listの構文
# 反復(for)で入力された値の合計を求める

number = [0, 0, 0, 0, 0]      # 要素を5つもつリストを宣言。
sum = 0

print('数値を入力してください。')
for i in range(5):            # i = 0,1,2,3,4で実行。
  number[i] = input(str(i + 1) + '個目') # 数値を入力してもらう。

for i in range(len(number)):  # リストの長さだけ繰り返す。
  sum += int(number[i])       # リストの要素を整数に変換しながら、変数『sum』に加えていく。

print('合計は', sum)    # 変数『sum』を文字列に変換して、表示。

関数の定義

関数はJavaでいうメソッドみたいなもの。
値を変えて同じような処理をする場合、何度も同じコードを書くと可読性も悪いし、めんどくさい。
だから同じような処理は関数(メソッド)として定義し、変動する値は定義した処理を呼び出す時にパラメータとして与える。
Javaとの違いは以下のとおり。
・Pythonではどこでも関数を書けるが、Javaでは必ずクラスの中に書く
・Pythonでは戻り値の設定や型など自由に書けるが、Javaではメソッドの最初に指定する
・Javaではmainメソッドから実行される
以下、Pythonの関数定義のコード例で確認。

practice.py 

# 関数の定義
# 再帰関数(自分自身を関数の中で繰り返し実行)：組合せ(nCr)の計算プログラム

# 組合せ nCr を求めるプログラム。「[n]C[r] = [n-1]C[r] + [n-1]C[r-1]」を利用する
# nは全体の個数、rは選ぶ個数、Cは組み合わせのこと、「n個の中からr個を順不同で選ぶ方法の数」を求める
# nCrはr=0またはr=nのとき1が戻り値となり、それ以外はn-1Cr + n-1Cr-1 が戻り値となる

def kumiawase(n ,r):    # 関数定義「kumiawase」が関数名、(n, r)はパラメータ
  if r == 0 or n == r:  # nCrはr=0またはr=nのとき
    return 1            # 1が戻り値
  else:                 # それ以外のとき
    return kumiawase(n - 1, r)  + kumiawase(n - 1, r - 1) # kumiawase関数を再度呼び出し

# ----▲ここまでが関数の定義▲----

n = input('整数を入力してください：')

for i in range(int(n)):   # inputされた値の回数だけ繰り返し
  print(n,'C',i,"=",kumiawase(int(n),i))  # kumiawase

7 C 6 = 7
