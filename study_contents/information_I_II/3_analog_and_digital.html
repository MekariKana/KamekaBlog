アナログとデジタル

アナログは手作業、手動でデジタルは機械的、自動っていうイメージがあった。でも定義としては違うみたい。
以下にアナログ、デジタルとはどういうことか、まとめる。

☻
アナログとデジタルの違い

アナログとデジタルの違いは「情報の表し方」。
時間や温度のような”量”を表現する場合、アナログは連続的な変化を表現できるのに対し、デジタルは段階的な変化を表現する。
アナログデータは連続的な変化を表すので、正確な量を表現できる。
デジタルデータは段階的な変化を表すので、近似値で表現される。
[例]体温計
アナログ：水銀体温計は体温を水銀の長さで表現するので、なめらかな体温変化を見れる。数字が書かれていないところで温度（水銀）が止まることもある。
デジタル：デジタル体温計は、段階的な温度（数字）で表現するので正確の値ではない。37.5度と表示されても、実際はそれより低かったり、高かったりする。

コンピュータはデジタル表現のみを扱う
・コンピュータが取り扱う文字、画像、音楽など全ての情報はデジタルで表現
・取り扱うデータの表現方法は0と1の2段階→この表現方法を「2進法」、表現された数字を「2進数」という
・文字、画像、音楽などの情報を決められたルールに沿って0と1で処理

コンピュータはどのように数字を読み取るのか？
コンピュータの中にあるIC（集積回路）という部品に電圧を流し、その電圧の高さによって0と1を区別している。
直流電圧が0V（0ボルト）なら0、5V（5ボルト）なら1。
一定の間隔で電圧を読み取ることで、0か1の並びを取得している。

情報量
・情報量の最小単位は1ビット
・1bitで表される情報量は0か1の2パターン
・8bitをまとめて扱う=1byte
・1byte(8bit)で表される情報量は256パターン(2の8乗)ある
・情報量は1024パターン(2の10乗)毎に単位が変わる

単位 単位が変わる量
bit 最小単位
B 1Byte = 8bit
KB 1KByte = 1024Byte
MB 1MByte = 1024KByte
GB 1GByte = 1024MByte
TB 1TByte = 1024GByte
PB 1PByte = 1024TByte

ここで素朴な疑問、なんで単位の区切りとなる数が1000じゃなくて1024なの？
これについてはChatGPTに質問、以下のように回答を得た。

1000は人間（10進数）にとってキリのいい数字

以下のように2進数の世界ではキリが悪く、メモリの管理やアドレス計算が非効率になる。
1000（10進数） = 1111101000（2進数）

1024がコンピュータ（2進数）の世界でキリがいい数字

1024（10進数） = 10000000000（2進数）

ちなみに1byte = 8bitなのに1024は11bitじゃん。8bit超えていいの？って少し混乱したので、こちらもChatGPTに質問。
データは8bitづつ扱うのに、データの大きさを表す単位は1024毎なの？

これは「データの最小単位（1byte）」と「記憶装置の管理単位（1024byte単位）」が別の役割を持ってるからなんです。
データの「細かい扱い」は1byteごと、でも「大きさの測定」は1024ごとにまとめた方が都合がいい（2進数的に）のです。

データは8bit（=1byte）ごとに扱う → 情報の基本単位

「8bit = 1byte」は、データをどの単位で扱うか（データのかたまり）の話。
・例：文字1つ → 1バイト（例：A = 01000001）
・CPUやメモリはこの1バイト単位でデータを読んだり書いたりする
= これらは「データの粒」の話

容量の単位（KB, MB）は、1024ごと → 管理・記録のための目安

・例：メモリが「8GBあります」って言ったら、8×1024×1024×1024バイトってこと
・なぜ1024かというと → 2進数で管理しやすいから！（2の乗数：2^10）
= これは「どれくらいの大きさをまとめて数えるか」の話

例えてみると・・・郵便局のイメージ

・1byte：封筒1通
・1024byte (1KB)：ダンボール1箱分くらい
データの中身は封筒1通ずつ（byte単位）だけど、倉庫では「ダンボール箱（1024通）」単位で数えてる
→ 「封筒は1通ずつ扱うけど、箱に詰める時は1024通ずつね」って感じ！

要はbitはただ数を表すための単位で、大きな数を表すにはbit数も多くなる。
1byte=8bitはデータの最小単位で、8bitづつをコンピュータは1つのデータとして処理している。
1024byteっていうのは容量をまとめて数える時の一塊（区切り）。コンピュータが理解しやすい数でまとめた。


デジタル表現の特徴

1）ノイズが混じっても修復できる
アナログの波形は連続的で細やに表現されている分ノイズが加わると修復できず、情報が変わってしまう。
デジタルの波形はコンピュータはある一定の電圧以上を1、それ以下を0と読み取るので元の波形に修復できる。

2）加工できる
文字、画像、音楽など全て0か1で表現されているため、修正・編集などが簡単にできる。
また、圧縮や暗号化しても簡単に元に戻せる。
※圧縮とは、一定のルールに従ってデータ量を小さくすること
※可逆圧縮は圧縮前後でデータが完全に同じに展開できる圧縮方法。文書ファイルやプログラムファイルは1bitたりとも変わってはいけないので、可逆圧縮を使用する。
※非可逆圧縮は画像や音声ファイルに使われる。人間にはわからないレベルで画質を落としたり、音楽の音域をカットする。データを書き換えるので、圧縮後のデータを元には戻せない。

3）さまざまな機器で取り扱える
例）音楽データ
アナログデータ：レコードやカセット
専用の再生機器が必要。レコードをカセット機器で聞くことはできない。
デジタルデータ：CD
音声データが一定のルールに従って0と1の並びで表現されているので、スマホ、PC、CDプレーヤーなど複数の機器で同じように再生できる。


以上〜 アナログの方が正確っていうけど、結局人間が認識できる範囲で情報が表現されていれば問題ないし、むしろデータ駆動型社会という考え方がある現在ではデータが劣化しないこと、壊れないことのほうが重要だったりするから、デジタルのほうが今は重宝されるんだなぁと理解。
でも脳みそには「手を動かす」ことが大事だし、個人的には人間の暖かみを感じる手作業の方が嬉しいこともあるので、ケースバイケースか。
