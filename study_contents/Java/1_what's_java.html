Javaの基本知識

Javaとは

Javaとはプログラミング言語の一種で、プログラミング言語とはコンピュータにしてほしいことを伝えるための言語。
私たちがプログラミング言語を使って書くプログラムをソースコードという。
しかしコンピュータ（CPU）が理解できる言語は機械語（2進数（0と1の信号））のみなので、機械語以外の言語は必ず機械語に変換される。
この変換する作業をコンパイル、コンパイルするためのツールを「コンパイラ」という。
コンパイラが作成した実行形式のファイルをOS（windows、Mac、Linuxなど）が動かす。

Javaの特徴

どの環境でも実行できる
実行形式のファイルはOSによって違うため、windowsのファイルをMacで使ったりはできない。そのため違うOSで動作するときは、そのOSの特性を考えてソースコードを書き直し、そのOS用のコンパイラでコンパイルする必要がある。 このように違う環境で動作できるようにすることを「移植する」という。
しかし、Javaの場合はそれぞれのOSの上に「Java仮想マシン（JVM）」という仮想コンピュータを配置し、このJVMの上でプログラムが動く仕組みになってる。
Javaのコンパイラは、ソースコードをJVMが理解できるコード（バイトコード）へコンパイルする。なのでJVMがある環境では、どこでもJavaのプログラムを動かすことができる。

オブジェクト指向
各機能や役割を持ったプログラムごとに分割して部品のように扱う。
このメリットは各機能が独立しているので、役割分担がしやすく大規模開発しやすいこと、部品の再利用がしやすいことが挙げられる。
