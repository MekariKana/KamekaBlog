横並び配置はflex

このHPを作成した際、一番重宝したのが "display: flex;" というcssの設定。
HomeページのUpdatingの各項目やその中の日付と更新内容の横並び、 profileページの経歴の画像と文字の横並び、 portfolioページの作品集の横並びなど、なにかと横並び場面で使用してきた。
今後もページ更新の際は多様しそうなので、 portfolioページを例としてメモしておく。

＜使い方＞
1. htmlで横並びさせたい要素をdivで囲む

index.html 

<div class="w_inner">
    <!--横配置したい要素をdivで囲む-->
    <div class="portfolio_block">

        <!--横に並べたい要素たち-->
        <div class="portfolio">
            <!--作品1-->
        </dix>

        <div class="portfolio">
            <!--作品2-->
        </dix>
        
        <div class="portfolio">
            <!--作品3-->
        </dix>

    </div>
</div>

2. cssで親要素<portfolio_block>クラスに "display: flex;" を指定する
3. divの中の子要素<portfolio>が横並びになるので、親要素<w_iiner>クラスのwidthに合わせて<portfolio>のwidthを設定する
※ここではjustify-content: space-betweenを使用しているため、等間隔に配置される

style.css 

/* 全体の横幅は1200pxで指定 */
.w_iiner {
    width: 1200px;
    margin: 0 auto;
}

/* .portfolioの親要素でdisplay: flex;を設定 */
.portfolio_block {
    display: flex; /* 子要素が横並びになる設定 */
    flex-wrap: wrap; /* 要素を折り返して表示 */
    justify-content: space-between; /* 両端揃えで均等に配置 */
    text-align: center;
}

/* 各portfolioブロックのスタイル */
.portfolio_block .portfolio {
    width: 380px; /* 3つが横に並べるサイズに設定 */
    text-align: center;
    margin-bottom: 30px;
}

これでportfolioページみたいな横配置になりました〜
